# WebSemantiqueApp

To run in dev mode : 

``` npm run dev ``` 
## Prerequisites

- Node.js (v14 or higher)
- npm or yarn

## Installation

```
npm install
```

## Build for production

```
npm run build
```

## Preview production build

```
npm run preview
```

## Project Structure

- `src/` - Source files
- `public/` - Static assets
- `dist/` - Production build output

## Technologies

- Vite
- Vue 3 (or your framework)
